# ZanderJi.github.io
本源码为腕上微聊php端源码.
注意:本源码默认域名为sayqz.top.

帮助:
1.将sayqz.top文件夹内的源码复制进您的服务器;
2.在"腕管Pro>应用设置>腕上微聊>服务器设置"中,将地址改为:
"http://您的域名/app/com.cn.awg.pro/talking",或"https://您的域名/app/com.cn.awg.pro/talking";
3.保存,即可运行腕上微聊.
